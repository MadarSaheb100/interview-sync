N1

Webhook: Receive S1 Input

n8n-nodes-base.webhook · v2
🎯
What it does: This is the entry point of the entire workflow. When Aditi clicks "Generate Optimized Schedule" on the S1 screen, the InterviewSync frontend makes a POST request to this webhook URL. The node receives the full payload — all 8 candidates, 12 interviewers, and 3 rooms — and passes it down the chain.
1
Add the node — search "Webhook" in the node picker

In the canvas, click the + button → search "Webhook" → select Webhook (not HTTP Request). This is always the first node.

2
Configure all fields in the node panel
FieldValue to setWhy
HTTP Method POST We're sending a full JSON body (candidates, interviewers, rooms). POST is required for body data.
Path interviewsync/s1/generate This becomes part of your webhook URL. No leading slash needed. n8n will prefix it with your instance URL automatically.
Response Mode Response Node Critical. This tells n8n to NOT auto-respond. Instead, the "Respond to Webhook" nodes (N8 / N9) send the final response. Without this, you'd get an empty response before scheduling completes.
Authentication None For hackathon / development. In production, set to "Header Auth" and add an API key.
⚠️
Response Mode = "Response Node" is mandatory. If you leave it as "Last Node" or "Immediately", the workflow will send an empty 200 before the schedule is generated. Aditi's frontend will receive nothing useful.

N2

Validate Input Payload

n8n-nodes-base.code · v2
🛡️
What it does: Before touching the scheduler, this node checks every field in the incoming payload. If a candidate is missing availableFrom, or an interviewer has no canInterview array, the workflow stops here and returns a specific error. This protects Aditi from a silent bad schedule.
1
Add a Code node → rename it "Validate Input Payload"

Click + after N1 → search "Code" → select it. Double-click the node title to rename it.

2
Set Language to "JavaScript" (default) — paste the code below
JavaScript · Code Node 
// ─────────────────────────────────────────────────────────
// NODE 2: Validate Input Payload
// Reads the POST body from the Webhook node.
// Returns status: 'VALID' or status: 'VALIDATION_FAILED'
// ─────────────────────────────────────────────────────────

const body = $input.first().json.body;
const errors = [];

// ── Top-level arrays must exist ───────────────────────────
if (!body.candidates || !Array.isArray(body.candidates) || body.candidates.length === 0) {
  errors.push('Missing or empty candidates array');
}
if (!body.interviewers || !Array.isArray(body.interviewers) || body.interviewers.length === 0) {
  errors.push('Missing or empty interviewers array');
}
if (!body.rooms || !Array.isArray(body.rooms) || body.rooms.length === 0) {
  errors.push('Missing or empty rooms array');
}

// ── Validate each candidate ───────────────────────────────
if (body.candidates) {
  body.candidates.forEach((c, i) => {
    if (!c.id)   errors.push(`Candidate[${i}] missing id`);
    if (!c.name) errors.push(`Candidate[${i}] missing name`);
    if (!c.role) errors.push(`Candidate[${i}] missing role`);
    if (!c.availableFrom || !c.availableTo) {
      errors.push(`Candidate[${i}] (${c.name || i}) missing availableFrom/availableTo. Use 24h format: "10:00"`);
    }
  });
}

// ── Validate each interviewer ─────────────────────────────
if (body.interviewers) {
  body.interviewers.forEach((iv, i) => {
    if (!iv.id)   errors.push(`Interviewer[${i}] missing id`);
    if (!iv.name) errors.push(`Interviewer[${i}] missing name`);
    if (!iv.canInterview || !Array.isArray(iv.canInterview)) {
      errors.push(`Interviewer[${i}] (${iv.name || i}) missing canInterview array`);
    }
    if (!iv.availableFrom || !iv.availableTo) {
      errors.push(`Interviewer[${i}] (${iv.name || i}) missing availableFrom/availableTo`);
    }
  });
}

// ── Validate each room ────────────────────────────────────
if (body.rooms) {
  body.rooms.forEach((r, i) => {
    if (!r.id)      errors.push(`Room[${i}] missing id`);
    if (!r.name)    errors.push(`Room[${i}] missing name`);
    if (!r.purpose) errors.push(`Room[${i}] missing purpose. Use: "senior", "technical", or "design_culture"`);
  });
}

// ── Return result ─────────────────────────────────────────
if (errors.length > 0) {
  return [{ json: { status: 'VALIDATION_FAILED', errors: errors } }];
}

return [{ json: { status: 'VALID', payload: body } }];
✅
Output shape: Either {"status":"VALID","payload":{...}} or {"status":"VALIDATION_FAILED","errors":["..."]}. The IF node (N3) reads the status field to decide which branch to take.

N5

Generate Optimized Schedule

n8n-nodes-base.code · v2
🧠
What it does: The core scheduling engine. For each of the 8 candidates, it assigns 3 rounds (R1 Technical, R2 Cross-functional, R3 Culture Fit) at 45 min each with 15 min gaps. It enforces: interviewer availability window, role expertise match, room availability (no double-booking), candidate hard blocks (prayer, flight stops), and raises seniority warnings. It never silently skips a round — unschedulable slots go into a separate unscheduledSlots[] array.
1
Add a Code node → connect from N3's TRUE output → rename "Generate Optimized Schedule"

This node receives $json.payload which contains the validated candidates, interviewers, and rooms arrays.

2
Paste the full scheduling engine code
JavaScript · Scheduling Engine (N5) 
// ─────────────────────────────────────────────────────────
// NODE 5: Generate Optimized Schedule
// Input:  $json.payload  (from N2 validation)
// Output: { status, summary, schedule[], warnings[], unscheduledSlots[] }
// ─────────────────────────────────────────────────────────

const { candidates, interviewers, rooms } = $input.first().json.payload;

// ── Time helpers ──────────────────────────────────────────
// toMins("10:30") → 630   (minutes since midnight)
const toMins = (t) => {
  const [h, m] = t.split(':').map(Number);
  return h * 60 + m;
};

// toTimeStr(630) → "10:30 AM"
const toTimeStr = (mins) => {
  const h = Math.floor(mins / 60);
  const m = mins % 60;
  const period = h >= 12 ? 'PM' : 'AM';
  const dh = h > 12 ? h - 12 : h === 0 ? 12 : h;
  return `${dh}:${String(m).padStart(2,'0')} ${period}`;
};

// ── Constants ─────────────────────────────────────────────
const ROUND_DURATION = 45;  // minutes per round
const GAP = 15;              // buffer gap between rounds

// ── 3 round types ────────────────────────────────────────
const ROUND_TYPES = [
  { id: 'R1', label: 'Technical / Domain',  matchBy: 'role'    },
  { id: 'R2', label: 'Cross-functional',     matchBy: 'role'    },
  { id: 'R3', label: 'Culture Fit',           matchBy: 'culture' },
];

// ── Room priority by round/role ───────────────────────────
// VP/Director → prefer Room A (Boardroom)
// Technical   → prefer Room B (has projector)
// Design/Culture → prefer Room C
const ROOM_PRIORITY = {
  senior:         ['A'],
  technical:      ['B', 'A'],
  design_culture: ['C', 'A'],
  culture:        ['C', 'B', 'A'],
};

// ── Booking registries (minute-level, prevents double-booking) ───
const roomBookings        = {};
const interviewerBookings = {};

const isRoomFree = (id, start, dur) => {
  for (let t = start; t < start + dur; t++) {
    if (roomBookings[`${id}|${t}`]) return false;
  }
  return true;
};
const bookRoom = (id, start, dur) => {
  for (let t = start; t < start + dur; t++) roomBookings[`${id}|${t}`] = true;
};
const isIVFree = (id, start, dur) => {
  for (let t = start; t < start + dur; t++) {
    if (interviewerBookings[`${id}|${t}`]) return false;
  }
  return true;
};
const bookIV = (id, start, dur) => {
  for (let t = start; t < start + dur; t++) interviewerBookings[`${id}|${t}`] = true;
};

// ── Hard block overlap check ──────────────────────────────
// Used for prayer break, any other candidate-side hard blocks
const overlapsHardBlock = (candidate, start, dur) => {
  if (!candidate.hardBlocks || candidate.hardBlocks.length === 0) return false;
  const end = start + dur;
  for (const blk of candidate.hardBlocks) {
    const bs = toMins(blk.from);
    const be = toMins(blk.to);
    if (!(end <= bs || start >= be)) return true;
  }
  return false;
};

// ── Find a free interviewer matching role + time ──────────
const findInterviewer = (role, roundType, start, dur) => {
  for (const iv of interviewers) {
    const ivFrom = toMins(iv.availableFrom);
    const ivTo   = toMins(iv.availableTo);
    if (start < ivFrom || (start + dur) > ivTo) continue;
    let matches = false;
    if (roundType.matchBy === 'culture') {
      matches = iv.canInterview.includes('All') || iv.canInterview.includes('Culture');
    } else {
      matches = iv.canInterview.includes(role) || iv.canInterview.includes('All');
    }
    if (!matches) continue;
    if (!isIVFree(iv.id, start, dur)) continue;
    return iv;
  }
  return null;
};

// ── Find a free room for this round ──────────────────────
const findRoom = (roundType, role, start, dur, ivSeniority) => {
  let roomType = 'technical';
  if (roundType.id === 'R3') roomType = 'culture';
  else if (role === 'Product Design') roomType = 'design_culture';
  if (ivSeniority === 'VP' || ivSeniority === 'Director') roomType = 'senior';
  const priority = ROOM_PRIORITY[roomType] || ROOM_PRIORITY['technical'];
  for (const letter of priority) {
    const room = rooms.find(r => r.id === letter);
    if (!room) continue;
    if (isRoomFree(room.id, start, dur)) return room;
  }
  return null;
};

// ── Main scheduling loop ──────────────────────────────────
const schedule       = [];
const warnings       = [];
const unscheduled    = [];

for (const cand of candidates) {
  const candFrom     = toMins(cand.availableFrom);
  const candTo       = toMins(cand.availableTo);
  const hardStop     = cand.hardStop ? toMins(cand.hardStop) : candTo;
  const candSchedule = {
    candidateId:   cand.id,
    candidateName: cand.name,
    role:           cand.role,
    rounds:         [],
  };

  let currentStart = candFrom;

  for (const roundType of ROUND_TYPES) {
    let scheduled = false;
    let attempts  = 0;

    while (!scheduled && attempts < 480) {
      const slotEnd = currentStart + ROUND_DURATION;

      // Exceeds availability or hard stop?
      if (slotEnd > candTo || slotEnd > hardStop) {
        warnings.push({
          candidateId:   cand.id,
          candidateName: cand.name,
          round:         roundType.id,
          issue:         `Cannot fit ${roundType.label} within candidate availability window`
        });
        break;
      }

      // Hits a hard block? (e.g. prayer break)
      if (overlapsHardBlock(cand, currentStart, ROUND_DURATION)) {
        currentStart++; attempts++; continue;
      }

      // Find interviewer
      const iv = findInterviewer(cand.role, roundType, currentStart, ROUND_DURATION);
      if (!iv) { currentStart++; attempts++; continue; }

      // Find room
      const room = findRoom(roundType, cand.role, currentStart, ROUND_DURATION, iv.seniority);
      if (!room) { currentStart++; attempts++; continue; }

      // All good — commit the booking
      bookRoom(room.id, currentStart, ROUND_DURATION);
      bookIV(iv.id, currentStart, ROUND_DURATION);

      // Seniority warning (if Tech Lead covers VP-slot)
      if (iv.seniority === 'TechLead' && roundType.id !== 'R3') {
        warnings.push({
          candidateId:   cand.id,
          candidateName: cand.name,
          round:         roundType.id,
          issue:         `Seniority note: ${iv.name} (TechLead) covering a VP-level slot. Aditi approval recommended.`
        });
      }

      candSchedule.rounds.push({
        roundId:          roundType.id,
        roundLabel:       roundType.label,
        startTime:        toTimeStr(currentStart),
        startMins:        currentStart,
        endTime:          toTimeStr(currentStart + ROUND_DURATION),
        endMins:          currentStart + ROUND_DURATION,
        interviewerId:    iv.id,
        interviewerName:  iv.name,
        interviewerRole:  iv.role || '',
        roomId:           room.id,
        roomName:         room.name,
        status:           'SCHEDULED',
      });

      currentStart = currentStart + ROUND_DURATION + GAP;
      scheduled = true;
    }

    if (!scheduled) {
      unscheduled.push({
        candidateId:   cand.id,
        candidateName: cand.name,
        round:         roundType.id,
        reason:        `No valid interviewer+room combination found within candidate window`
      });
    }
  }

  schedule.push(candSchedule);
}

// ── Build final output ────────────────────────────────────
const totalExpected  = candidates.length * ROUND_TYPES.length;
const totalScheduled = schedule.reduce((s, c) => s + c.rounds.length, 0);

return [{
  json: {
    status:          unscheduled.length === 0 ? 'SCHEDULE_COMPLETE' : 'SCHEDULE_PARTIAL',
    summary: {
      totalCandidates:    candidates.length,
      totalExpected,
      totalScheduled,
      totalWarnings:      warnings.length,
      totalUnscheduled:   unscheduled.length,
      coveragePercent:    Math.round((totalScheduled / totalExpected) * 100),
    },
    schedule,
    warnings,
    unscheduledSlots: unscheduled,
    generatedAt:      new Date().toISOString(),
  }
}];
✅
Output shape: { status, summary{}, schedule[], warnings[], unscheduledSlots[], generatedAt }
⚠️
Edge cases handled inside this code:
• Prayer break — overlapsHardBlock() skips the 1:00–1:30 PM window for Priya
• Hard stop — Sneha (3:30 PM) and Vikram (3:00 PM) get no slot that overruns
• Interviewer double-booking — minute-level bookIV() registry
• Room double-booking — minute-level bookRoom() registry
• Seniority mismatch — warning added, not a block
• No slot found — added to unscheduledSlots[], never silently dropped

JavaScript · Draft Notifications (N6) 
// ─────────────────────────────────────────────────────────
// NODE 6: Draft Notifications
// Input:  $json from N5 (schedule, warnings, unscheduledSlots)
// Output: same $json + notifications[] array (all DRAFT status)
// ─────────────────────────────────────────────────────────

const { schedule, warnings, unscheduledSlots, summary } = $input.first().json;
const notifications = [];

// ── 1. Candidate notifications ─────────────────────────
for (const cand of schedule) {
  if (cand.rounds.length === 0) continue;

  const roundLines = cand.rounds.map(r =>
    `• ${r.roundId} (${r.roundLabel}): ${r.startTime} – ${r.endTime} with ${r.interviewerName} in ${r.roomName}`
  ).join('\n');

  notifications.push({
    to:        cand.candidateName,
    type:      'CANDIDATE',
    id:        cand.candidateId,
    channel:   'email',
    subject:   'Your Interview Schedule — NovaBuild Super Day',
    body:      `Hi ${cand.candidateName.split(' ')[0]},\n\nWe're excited to have you join us for the NovaBuild Super Day! Here is your personalised schedule:\n\n${roundLines}\n\nPlease arrive 10 minutes before your first round.\n\nBest of luck!\nAditi Sharma, HR Lead — TechScale`,
    status:    'DRAFT',
  });
}

// ── 2. Interviewer notifications (grouped by interviewer) ─
const ivSlots = {};
for (const cand of schedule) {
  for (const round of cand.rounds) {
    if (!ivSlots[round.interviewerId]) {
      ivSlots[round.interviewerId] = { name: round.interviewerName, slots: [] };
    }
    ivSlots[round.interviewerId].slots.push({
      candidateName: cand.candidateName,
      role:          cand.role,
      round:         round.roundId,
      roundLabel:    round.roundLabel,
      startTime:     round.startTime,
      endTime:       round.endTime,
      roomName:      round.roomName,
    });
  }
}

for (const [ivId, ivData] of Object.entries(ivSlots)) {
  const slotLines = ivData.slots.map(s =>
    `• ${s.startTime}–${s.endTime}: ${s.candidateName} (${s.role}) — ${s.roundLabel} — ${s.roomName}`
  ).join('\n');

  notifications.push({
    to:        ivData.name,
    type:      'INTERVIEWER',
    id:        ivId,
    channel:   'email',
    subject:   'Your Panel Schedule — NovaBuild Super Day',
    body:      `Hi ${ivData.name.split(' ')[0]},\n\nThank you for being part of the NovaBuild Super Day panel! Your assigned slots:\n\n${slotLines}\n\nCandidate profiles are available in Lever. Please review before each session.\n\nThank you,\nAditi Sharma, HR Lead — TechScale`,
    status:    'DRAFT',
  });
}

// ── 3. Warning alert → shown on Aditi's dashboard ────────
if (warnings.length > 0) {
  const lines = warnings.map(w => `• [${w.candidateId}] ${w.candidateName} — ${w.round}: ${w.issue}`).join('\n');
  notifications.push({
    to:      'Aditi Sharma',
    type:    'HR_ALERT',
    channel: 'dashboard',
    subject: `${warnings.length} Schedule Warning(s) — Review Required`,
    body:    `Seniority / constraint warnings raised during schedule generation:\n\n${lines}\n\nPlease confirm or override each flagged slot.`,
    status:  'DRAFT',
  });
}

// ── 4. Critical alert if any rounds couldn't be scheduled ─
if (unscheduledSlots.length > 0) {
  const lines = unscheduledSlots.map(u => `• ${u.candidateName} — ${u.round}: ${u.reason}`).join('\n');
  notifications.push({
    to:      'Aditi Sharma',
    type:    'HR_CRITICAL',
    channel: 'dashboard',
    subject: `CRITICAL: ${unscheduledSlots.length} Round(s) Could Not Be Scheduled`,
    body:    `These rounds could not be scheduled due to interviewer/room constraints:\n\n${lines}\n\nManual intervention required before Super Day begins.`,
    status:  'DRAFT',
  });
}

// ── Pass everything forward ───────────────────────────────
return [{
  json: {
    ...$input.first().json,   // keep everything from N5
    notifications,
    notificationCount: notifications.length,
  }
}];
